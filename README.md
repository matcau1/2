# CRM для промышленного холодильного сервиса

Простая CRM-система на JavaScript + React для ведения заказчиков (компаний), сотрудников и файлов.

## Что реализовано

- CRUD заказчиков: создание, редактирование, удаление.
- Поля заказчика: название, ИНН, адрес, контакты, статус.
- Карточка заказчика с:
  - загрузкой логотипа,
  - загрузкой файлов (счета, договоры, акты, фото и т.д.),
  - CRUD сотрудников (имя, фамилия, телефон, e-mail).
- Хранение данных в **MySQL**.
- Ленивые загрузки (по мере надобности):
  - при старте загружается только список заказчиков,
  - детальная карточка (логотип, файлы, сотрудники) подгружается только при открытии выбранного заказчика.

## Настройка окружения

1. Скопируйте пример переменных окружения:

```bash
cp .env.example .env
```

2. Заполните значения MySQL в `.env`.

## Запуск

```bash
npm install
npm run dev
```

Команда поднимает одновременно:
- фронтенд (Vite) на `http://localhost:5173`,
- API-сервер на `http://localhost:3001`.

## Дополнительные команды

```bash
npm run dev:client
npm run dev:server
npm run build
```

## База данных

При старте `server.js` автоматически создаёт таблицы (если их ещё нет):
- `Customers`
- `CustomerFiles`
- `CustomerEmployees`
